# Embedded Programming – Task 3

This repository contains solutions for Embedded Programming Task 3
using Wokwi simulator.

## Folder Structure

Embedded_Task_3 contains 8 separate Wokwi projects.

Each Wokwi project includes:
- diagram.json → hardware wiring and components
- main.c or main.py → embedded application code

## Notes
- C programs are written for Raspberry Pi Pico.
- MicroPython programs are written for ESP32.
- Each folder represents one independent embedded task.
